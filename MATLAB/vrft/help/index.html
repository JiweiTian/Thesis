Welcome to the Virtual Reference Feedback Tuning Toolbox guide.

 

VRFT ("Virtual Reference Feedback Tuning") is a general methodology for the design of feedback controllers when the plant is unknown. Its main features are:

  • it is a direct method (no model identification of the plant is needed);
  • it can be applied using a single set of data collected from the plant with no need for specific experiments nor iterations.
    What it is used for:

    VRFT permits to tune a 1 or 2 degree of freedom linear controller within a specified class on the basis of a single set of input/output data collected from an experiment on the plant.

     

     
    How it is used:

    The control specifications are assigned via a model-reference discrete time transfer function. Given a single set of input/output data (either collected in open-loop or in closed-loop), the VRFT method can be applied in order to obtain the controller that satisfies the assigned specifications.
     
    How it works:

    The "design engine" of VRFT is an solver of a model-reference optimization problem. Lack of knowledge of the plant transfer function is compensated for via I/O measurements.
     

     

VRFT Toolbox - Version 1.0

 

VRFT1_ry: Design a 1 degree of freedom (1 d.o.f.) linear controller so as to match the r(t) to y(t) closed-loop transfer function with the model reference Mr (see Figure 1).

Figure 1: 1 degree of freedom control scheme.

 

VRFT1_dy: Design a 1 d.o.f. linear controller so as to match the d(t) to y(t) output sensitivity with the model reference Md (see Figure 1).

VRFT1_ry_ru: Design a 1 d.o.f. linear controller so as to match the r(t) to y(t) closed-loop transfer function with the model reference Mr and the r(t) to u(t) input sensitivity with the model reference Mu (see Figure 1).

VRFT1_dy_du: Design a 1 d.o.f. linear controller so as to match the d(t) to y(t) closed-loop transfer function with the model reference Md and the d(t) to u(t) input sensitivity with the model reference Mu (see Figure 1).

VRFT2_ry_dy: Design a 2 d.o.f. linear controller so as to match the r(t) to y(t) transfer function and the d(t) to y(t) transfer function (see Figure 2).
