Chosing a reference model

The requirements are \(T_s < 1s\) and \(S_\% < 25\%\). The overshoot requirement imposes \(\zeta < 0.4\) whilst the settling time can be approximated with (cf: previous post):

\begin{equation*} T_s \approx \frac{5}{\zeta\omega_n} = \frac{12.5}{\omega_n} \end{equation*}

Thus the required bandwidth is:

\begin{equation*} \omega_n = \frac{12.5}{T_s} = 12.5 \quad rad/s \end{equation*}

The corresponding second order model is obtained with:

[num, den] = ord2(12.5, 0.4);
RefModel = tf(num, den);
\begin{equation*} M_R = \frac{1}{s^2 + 10s + 156.2} \end{equation*}
