Discrete systems theory

17 marzo 2016

Definitions

Consider the general form of a discrete time system:

\begin{equation*} \begin{aligned} x\left( k+1 \right) &= f\left( x\left( k \right), u\left( k \right), k \right) \\ y(k) &= g\left( x\left( k \right), u\left( k \right), k \right) \end{aligned} \end{equation*}

Strictly proper systems

If the output equation of the system can be written as \(y(k) = g\left( x\left( k \right), k \right)\) then the output variable des not depend directly on the input and the system is strictly proper. The input must propagate through the state before affecting the output.

Time Invariant systems (Stationary)

If the functions \(f\) and \(g\) do not explicitely depend on the time variable $k$ then the system is time invariant and can be rewritten as:

\begin{equation*} \begin{aligned} x\left( k+1 \right) &= f\left( x\left( k \right), u\left( k \right) \right) \\ y(k) &= g\left( x\left( k \right), u\left( k \right) \right) \end{aligned} \end{equation*}

Reachability (Controllability)

A generic state \(\widetilde{x}\) is reachable if there exist an instant of time \(\widetilde{k}\) to \(\widetilde{k} - 1\) such that the forced state of the system \(\widetilde{x}_f(k)\) with \(0 \leq k \leq \widetilde{k}\) generated by \(\widetilde{u}\) produces \(\widetilde{x}_f(k) = \widetilde{x}\) .

The Reachability Matrix is:

\begin{equation*} \underline{M}_R = \begin{bmatrix}B \quad AB \quad A^2B \quad ... \quad A^{n-1}B\end{bmatrix} \end{equation*}

Observability

A state \(\widetilde{x} \neq 0\) is said to be non observable if, for any \(\widetilde{k} > 0\) the free motion of the system \(\widetilde{y}_l(k) = 0\) with \(0 \leq k \leq \widetilde{k}\).

The Observability Matrix is:

\begin{equation*} \underline{M}_O = \begin{bmatrix} C \\ CA \\ CA^2 \\ \vdots \\ CA^{n-1} \end{bmatrix} \end{equation*}

Transfer function

The general form of the transfer function for a discrete time system is:

\begin{equation*} G(z) = C(zI - A)^{-1}B + D \end{equation*}

By construction \(C(zI - A)^{-1}B\) is a rational function in $z$ where the denominator is of degree $n$ and the numerator's degree is at most \(n-1\). This is the form of a strictly proper system (\(D = 0\)).

If the system is not proper \(G(z)\) is still a rational function in z but both the numerator and the denominator have degree \(n\).

If the numerator and the denominator have common roots they must cancel each other out to produce a rational function whose degree is \(\nu \lt n\)

The transfer function can be written as:

\begin{equation*} G(z) = \frac{N_G(z)}{D_G(z)} = \frac{\beta_{\nu}z^{\nu} + \beta_{\nu-1}z^{\nu-1} + ... + \beta_1z + \beta_0}{\alpha_{\nu}z^{\nu} + \alpha_{\nu-1}z^{\nu-1} + ... + \alpha_1z + \alpha_0} \end{equation*}

Without losing in generality we can assume \(\alpha_{\nu} = 1\).

Time Delay

A time discrete time delay is described by the following input-output form:

\begin{equation*} y(k) = u(k-h) \Leftrightarrow Y(z) = z^{-h}U(z) \end{equation*}

Thus the tranfer function associated with the delay component is \(G(z) = z^{-h}\)

Considering a system where the action of the input is delayed by \(h\) ticks such as:

\begin{equation*} \begin{aligned} v(k) &= u(k-h)\\ x(k+1) &= Ax(k) + Bu(k) \\ y(k) &= Cx(k) + Du(k) \end{aligned} \end{equation*}

It can be show that the transfer function is:

\begin{equation*} G(z) = z^{-h}G'(z) \qquad, \qquad G'(z) = C(zI - A)^{-1}B + D \end{equation*}

Frequency Response

The frequency response of the transfer function is calculated as:

\begin{equation*} G(e^{j\theta}) = C(e^{j\theta}I - A)^{-1}B + D \qquad, \qquad \theta \in \left[0, \pi \right] \end{equation*}

This is simply a restriction of the transfer function to the semi-circumference defined by the complex points with positive or null immaginary part and unit modulus excluding those points that are also poles of \(G(z)\).

Settling Time

Consider a first order system:

\begin{equation*} G(z) = \frac{\mu(1-p)}{z-p} \qquad, \qquad p \neq 0 \end{equation*}

The step response of this system is:

\begin{equation*} y(k) = \mu(1-p^k) \qquad, \qquad k \geq 0 \end{equation*}

And the steady state value is \(y_{\infty} = \mu\).

The settling time of the system is the time it takes to output variable to enter the range \(\left[y_{\infty}(1 - 0.01\epsilon), \quad y_{\infty}(1 - 0.01\epsilon) \right]\)

\begin{equation*} \begin{alignedat}{3} & y_{\infty}(1 - 0.01\varepsilon) && \leq (1-p^k) &&< y_{\infty}(1+ 0.01\varepsilon) \\ &\frac{y_{\infty}}{\mu}(1 - 0.01\varepsilon) && \leq 1-p^k &&< \frac{y_{\infty}}{\mu}(1+ 0.01\varepsilon) \\ & (1 - 0.01\varepsilon) && \leq 1-p^k &&< (1+ 0.01\varepsilon) \\ & -0.01\varepsilon && \leq p^k &&< 0.01\varepsilon \\ \end{alignedat} \end{equation*}

Which can be traslated into the following requirement:

\begin{equation*} k \cdot ln(p) < ln(0.01\varepsilon) \qquad \Rightarrow \qquad k < \frac{ln(0.01\varepsilon)}{p} \end{equation*}

Thus, the settling time is:

\begin{equation*} T_{a\epsilon} = \lceil \frac{ln(0.01\varepsilon)}{ln(p)} \rceil \end{equation*}

Consider now a second order system with complex conjugate poles:

\begin{equation*} G(s) = \frac{\mu \omega^2_n}{s^2 + 2\zeta\omega_ns + \omega^2_n} \end{equation*}

This system has ann oscillatory behaviour and the peaks of the oscillations lie on 2 decaying exponentials:

\begin{equation*} y_{max}(t) = 1 + e^{-\zeta\omega_nt} y_{min}(t) = 1 - e^{-\zeta\omega_nt} \end{equation*}

Let \(\bar{t}\) be the instant in which the function \(y_{max}\) and \(y_{min}\) enter the range \(\left[ y_{\infty}(1 - 0.01\epsilon), \quad y_{\infty}(1 - 0.01\epsilon) \right]\). To determine \(\bar{t}\) we simply impose:

\begin{equation*} \begin{aligned} 1 + 0.01\varepsilon &= 1 + e^{-\zeta\omega_n\bar{t}} \\ 0.01\varepsilon &= e^{-\zeta\omega_n\bar{t}} \\ \bar{t} &= -\frac{1}{\zeta\omega_n} ln(0.01\varepsilon) \approx \frac{4.6}{\zeta\omega_n} \end{aligned} \end{equation*}
